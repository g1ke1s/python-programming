Industrial Python Development

This repository contains coursework for the Industrial Python Development course, focused on writing production-ready Python code.

The course covers:

Unit testing with pytest, TDD, debugging, and CI/CD basics

Building command-line tools with argparse, code quality checks, and test coverage

Working with Unicode and encodings, STDIN/STDOUT, and console testing

Logging configuration in Python (including YAML-based setups)

Mocking external dependencies for reliable testing

Web automation and parsing using requests, lxml, and BeautifulSoup

Core design patterns (Factory, Strategy, Composite, Decorator)

Developing and testing a Flask web service

Deployment and monitoring with Gunicorn, Nginx, Graphite, and Grafana

The repository demonstrates practical Python engineering skills used in real-world software development.
